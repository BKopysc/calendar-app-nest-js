# Clean Calendar API

## Description
Simple nest.js app for research project. Inlcudes backend implementation for calendar system.

## Documentation

Check [documentation folder](https://github.com/BKopysc/calendar-app-nest-js/tree/master/api-documentation)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
